Projeto Calculadora Python

Arquivos:

calculadora.py -> código em python
executar.sh -> script executável
README.md -> documentação


Como executar o .sh

chmod +x executar.sh

./executar.sh


Explicação do código Python

O programa pede dois números.
Depois mostra um menu.
O usuário escolhe a operação.
O programa calcula e mostra o resultado.


Explicação do arquivo .sh

O arquivo executar.sh serve para executar o python automaticamente.

Ele usa:

python3 calculadora.py
